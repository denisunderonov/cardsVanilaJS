# Карточки товаров

Приложение для отображения карточек товаров с возможностью фильтрации и сортировки.

## Технологии

- HTML5, CSS3, Vanilla JavaScript
- Flexbox для раскладки карточек
- LocalStorage для сохранения данных
- Адаптивная верстка

## Функционал

- Отображение карточек товаров
- Фильтрация по категориям
- Сортировка по цене
- Добавление в избранное
- Мобильная версия
